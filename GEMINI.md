# Instrucciones para Gemini

- Cuando identifiques problemas, decisiones o acciones que requieran intervención obligatoria por parte del humano, **DEBES** añadir una tarea en la sección `## 🛑 Tareas Bloqueantes (Humano)` del archivo `tareas_pendientes.md`.
- No asumas decisiones críticas ni realices acciones fuera de tu alcance. Fomenta que el humano desbloquee estas tareas antes de continuar con integraciones dependientes.
- Asegúrate de actualizar el estado de esta sección a medida que el humano resuelva los bloqueos.
- Al completar tareas o implementar cambios funcionales, **DEBES** encargarte de hacer los commits correspondientes y subir los cambios (push) a GitHub de forma autónoma. Esto asegura que las versiones estén disponibles para el resto del equipo sin que el usuario tenga que subirlas manualmente.
